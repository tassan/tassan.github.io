# Tassan's Blog

Welcome to **Tassan's Blog**! This is my personal space for sharing thoughts, tutorials, and articles in a sleek, dark-themed interface inspired by Evangelion EVA 01. Posts are written in Markdown and dynamically rendered on the site.

---

## 🔥 Features

- **Dark Theme:** A visually striking theme inspired by EVA 01's iconic colors.
- **Markdown Support:** Write posts in simple Markdown, and they’ll be rendered beautifully.
- **Dynamic Updates:** Add your Markdown posts to the `/posts` folder, and they’ll automatically appear on the blog.

---

## 🗂️ Folder Structure

```
tassan.github.io/
├── index.html                # Main blog page
├── css/
│   └── styles.css            # Styles for the blog
├── js/
│   └── markdown-renderer.js  # JavaScript to render Markdown posts
├── posts/
│   └── example.md            # Markdown blog posts go here
└── assets/
    └── images/               # Any additional assets (e.g., images) go here
```

---

## 🚀 Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/tassan/tassan.github.io
   ```
2. Add your Markdown posts to the `/posts` folder.
3. Open `index.html` in your browser to view your blog.
